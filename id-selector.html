This heading for ID selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.

In CSS, selectors declare which part of the markup a style applies to by matching tags and attributes in the markup itself. elements specified by attribute, in particular: id: an identifier unique within the document, denoted in the selector language by a hash prefix e.g. #id.
